Accessibility
Declaration of conformity

Compliance status

This website currently complies with the double AA level of the WCAG 2.1 standard.

Technologies

The site is built using HTML5, with its semantic tags, CSS3 and JavaScript. In addition, it uses Bootstrap 5.2 and Highcharts libraries, making use of all its accessible functionalities. It also uses the AOS library for animations, which can be disabled if the user wishes.

Most relevant improvements

  • All videos have the option to activate subtitles in the most common languages, using the YouTube API.
  • The graphics can be listened to by means of a suitable representation, both in the form of a description and a song, depending on the type.
  • The page can be navigated and read in full using a screen reader.

Preparation of the accessibility statement

This accessibility statement was prepared on January 5, 2023 through a self-assessment conducted by the site development team.
